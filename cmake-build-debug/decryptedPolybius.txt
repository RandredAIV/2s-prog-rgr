Массивы. Основные алгоритмы обработки массивов
Вернемся к подробному изучению массивов в С/С++. Как уже говорилось, массив объявляется следующим образом:
тип_данных имя_массива[количество элементов в массиве];
Причем, в С/С++ нумерация элементов массива начинается с нуля, поэтому если количество элементов в массиве равно ?, то индекс последнего
элемента равен ?.
Размер массива может явно не указываться. Например:
int p[]={2, 4, 6, ?, 1};
p[0]=2, p[1]=4, p[2]=6, p[3]=?, p[4]=1.
Количество элементов в массиве в этом случае можно определить так:
n=sizeof(p)/sizeof(int);
В результате следующего объявления массива
int M[6]={5, 3, 2};
будет создан массив из шести элементов. Первые три элемента получат
инициализированные значения. Значения остальных будут либо неопределенными, либо равны нулю, если массив внешний или статический.
