# ОПИСАНИЕ ПРОЕКТА
# ==============================================================

Данная расчётно-графическая работа посвящена разработке консольного
приложения для шифрования и дешифрования текстовой информации.

Программа реализует три классических криптографических алгоритма:

1) Двойная табличная перестановка
2) Квадрат Полибия
3) Тарабарская грамота

Проект реализован на языке программирования C++ с использованием
стандартной библиотеки STL и модульной архитектуры
(.cpp и .h файлы подключаются через main.cpp).

ВАЖНО:
Точкой входа является файл main.cpp.
Именно он подключает все остальные модули через директивы #include.

Программа работает в консольном режиме и поддерживает:

- Авторизацию пользователя
- Выбор алгоритма шифрования
- Шифрование текста
- Дешифрование текста
- Чтение данных из файла
- Запись результатов в файл
- Поддержку русского и английского алфавита
- Поддержку верхнего и нижнего регистра
- Обработку ошибок и исключений

# ==============================================================
# PROJECT DESCRIPTION
# ==============================================================

This course project is dedicated to the development of a console
application for text encryption and decryption.

The program implements three classical cryptographic algorithms:

1) Double Transposition Cipher
2) Polybius Square Cipher
3) Tarabar Cipher

The project is written in C++ using STL
and modular architecture.

IMPORTANT:
The entry point of the program is main.cpp.
All other modules are connected through it.

The application supports:

- User authentication
- Cipher selection
- Text encryption
- Text decryption
- File input
- File output
- Russian and English alphabets
- Uppercase and lowercase letters
- Error handling

# ==============================================================
# HOW TO BUILD
# ==============================================================

Navigate to project directory:

cd 2s-prog-rgr

Compile using main.cpp as entry point:

g++ main.cpp password.cpp table.cpp tarabar.cpp PolybiusCipher.cpp -o main

Static build (optional):

g++ main.cpp password.cpp table.cpp tarabar.cpp PolybiusCipher.cpp -o main -static

Executable file will be:

main.exe (Windows)
or
main (Linux/macOS)

# ==============================================================
